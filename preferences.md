# Préférences famille

## Adultes
- François n'aime pas les brocolis (les enfants oui)
- Les adultes apprécient les poireaux
- JAMAIS de beurre doux, uniquement du demi-sel

## Enfants
- Pas de soupe (ils ne voudront pas)
- Gnocchis : validé
- Croque-monsieur : validé
- Crêpes : validé
- Pizza maison : validé
- Steak-frites : validé

## Principes retenus
- Le soir en semaine, il faut aller vite (pas de cuisson longue type poulet rôti ou purée maison)
- Pas obligatoire d'avoir des féculents à chaque dîner
- On peut différencier les plats adultes / enfants quand c'est simple
- La ratatouille avec du riz sans protéine, ça passe pour un soir
- On ne planifie que les plats de résistance, pas les desserts
- Le riz doit toujours être du basmati
- Favoriser les produits pas chers
- Prendre les grosses quantités quand le prix au kg/L est meilleur
- Pas de friteuse, frites au four uniquement
- Légumes toujours FRAIS (jamais surgelés), prévenir si pas trouvé en frais
- Poireaux : toujours en botte frais (pas de rondelles surgelées)
- Salade : mâche de préférence (pas batavia)
- Pizza maison : prendre une pâte à pizza toute faite (pas farine + levure)
- Porc : filet mignon plutôt que rôti de porc

## Marques et produits préférés
- Haricots verts : Cassegrain en boîte extra-fins, 2 boîtes de 110g
- Lessive : Le Chat Bébé Peaux Sensibles, 2 bidons
- Gnocchis : Lustucru Extra Fromage
- Bière : Pelforth Blonde 20x25cL, 2 packs
- Couscous : Ferrero (pas SIMPL)
- Lait : SIMPL demi-écrémé, acheter 6 briques
- Café : Nescafé Spécial Filtre L'Original (recharge 150g), acheter x4
- Vin rosé : Gérard Bertrand Côte des Roses
- Purée de piments : Carrefour Sensation
- Pom'Potes : prendre le gros format (32 gourdes)
- Goûters enfants : Ourson LuLu (LU)
- Petit déjeuner : Croissants Pasquier x2 + Brioche tranchée Carrefour Classic
- Coriandre fraîche : herbe habituelle (2 bottes)
- Papier toilette : Lotus Aqua Tube x24
- Mouchoirs : Carrefour Soft 3 épaisseurs

## Plats validés
- Saumon crème légère + légumes
- Poulet rapide + couscous + carottes cuites
- Cabillaud pané + riz + courgettes
- Ratatouille + riz (sans protéine)
- Pizza maison tomates/champignons/jambon
- Filet mignon de porc + gratin dauphinois (dimanche midi)
- Crêpes salées/sucrées (dimanche soir)
